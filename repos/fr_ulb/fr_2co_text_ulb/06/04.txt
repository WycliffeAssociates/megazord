\v 4 Mais plutôt, à tous égards, nous avons fait preuve de vrais serviteurs de Dieu, dans l'endurance, dans l'affliction, dans la détresse, dans les épreuves, dans les coups, \v 5 dans les emprisonnements, dans les émeutes, dans les travaux durs, dans les nuits privées de sommeil, dans la faim, \v 6 dans la pureté, dans la connaissance, dans la patience, dans la gentillesse, dans l'Esprit Saint, dans l'amour sincère. \v 7 Nous sommes ses serviteurs  dans la parole de vérité, dans la puissance de Dieu, ayant les armure de justice pour les mains droite et gauche
